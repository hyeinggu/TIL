# HTML

**HTML**은 **웹페이지의 구조와 내용을 만들기 위한 마크업 언어**이다.

## 특징

- 웹페이지의 **구조와 내용을 표현**한다.
- **태그**를 사용해 문서를 작성한다.
- 대부분의 태그는 **여는 태그와 닫는 태그**로 구성된다.
- 웹 브라우저가 HTML 코드를 읽어 화면에 표시한다.

## 구조 태그

HTML 문서는 일정한 기본 구조로 이루어져 있다.

- **!DOCTYPE html**: HTML 문서라는 것을 선언
- **html**: 전체 HTML 문서
- **head**: 문서 정보(제목, 설정 등)
- **body**: 실제 화면에 보이는 내용

## 주요 태그

- **h1~h6**: 숫자가 작을수록 큰 제목
- **p**: 문단
- **br**: 줄바꿈
- **a**: 링크
- **img**: 이미지
- **ul/ol/li**: 리스트
- **div**: 영역
- **span**: 부분 영역
- **form**: 입력폼
- **input**: 입력
- **button**: 버튼

## 시맨틱 태그

**시맨틱** = **의미가 있는 태그**

- **header**: 웹페이지 **상단**
- **nav**: **네비게이션 메뉴**
- **main**: 페이지 **핵심 내용**
- **section**: 주제별 **내용 구역**
- **aside**: 광고, 추천글 같은 **보조 정보**
- **article**: 독립 콘텐츠
- **footer**: 페이지 **맨 아래**

## 역할

HTML은 웹 개발에서 **가장 기본적인 역할**을 한다.

- HTML: 웹페이지 **구조**
- CSS: 웹페이지 **디자인**
- JavaScript: 웹페이지 **동작과 기능**

## HTML 작성 프로그램

HTML 코드는 보통 VSCode 같은 코드 편집기를 사용해 작성한다.
작성한 HTML 파일을 웹 브라우저에서 열면 웹페이지 형태로 확인할 수 있다.
