# HTML

**HTML**은 **웹페이지의 구조와 내용을 만들기 위한 마크업 언어**이다.

## 특징

- 웹페이지의 **구조와 내용을 표현**한다.
- **태그**를 사용해 문서를 작성한다.
- 대부분의 태그는 **여는 태그와 닫는 태그**로 구성된다.
- 웹 브라우저가 HTML 코드를 읽어 화면에 표시한다.

## 역할

HTML은 웹 개발에서 **가장 기본적인 역할**을 한다.

- HTML: 웹페이지 **구조**
- CSS: 웹페이지 **디자인**
- JavaScript: 웹페이지 **동작과 기능**

## HTML 작성 프로그램

HTML 코드는 보통 VSCode 같은 코드 편집기를 사용해 작성한다.
작성한 HTML 파일을 웹 브라우저에서 열면 웹페이지 형태로 확인할 수 있다.
