# CSS

**CSS**는 **웹페이지의 디자인과 스타일을 꾸미는 스타일 시트 언어**이다.

웹페이지의 글자 색, 크기, 배경, 정렬, 여백 등을 설정하여 **웹페이지의 외형을 디자인하는 역할**을 한다.
보통 **HTML**과 함께 사용하여 웹페이지를 만든다.

## 역할

- 글자 색과 크기 설정
- 배경색이나 배경 이미지 설정
- 요소의 위치와 정렬 조정
- 여백과 테두리 설정
- 웹페이지 전체 디자인 관리

## 기본 구조

**선택자**와 **속성**으로 이루어진다.

- **선택자**: 스타일을 적용할 HTML 요소
- **속성**: 적용할 디자인 내용
- **값**: 속성에 설정할 값

## 사용 방법

- **인라인 스타일**: HTML 태그 안에 직접 작성
- **내부 스타일**: HTML 파일의 style 태그 안에 작성
- **외부 스타일**: 별도의 CSS 파일을 만들어 연결

## CSS와 웹 개발

웹 개발에서는 보통 역할을 나누어 사용한다.

- **HTML** → 웹페이지 구조
- **CSS** → 웹페이지 디자인
- **JavaScript** → 웹페이지 기능

이 세 가지가 함께 사용되어 하나의 웹사이트가 만들어진다.

## CSS 작성 프로그램

CSS 코드는 보통 VSCode 같은 코드 편집기를 이용해 작성한다.
작성한 CSS는 HTML 파일과 연결하여 웹 브라우저에서 확인할 수 있다.
