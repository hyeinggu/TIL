# CSS

**CSS**는 **웹페이지의 디자인과 스타일을 꾸미는 스타일 시트 언어**이다.

웹페이지의 글자 색, 크기, 배경, 정렬, 여백 등을 설정하여 **웹페이지의 외형을 디자인하는 역할**을 한다.

보통 **HTML**과 함께 사용하여 웹페이지를 만든다.

## 역할

- 글자 색과 크기 설정
- 배경색이나 배경 이미지 설정
- 요소의 위치와 정렬 조정
- 여백과 테두리 설정
- 웹페이지 전체 디자인 관리

## 기본 구조

**선택자(Selector)**+**속성(Property)**+**값(Value)** 구조로 작성한다.

- **선택자(Selector)**: 스타일을 적용할 HTML 요소
- **속성(Property)**: 적용할 디자인 내용
- **값(Value)**: 속성에 설정할 값

## Selector (선택자)

1. **태그 선택자**: 특정 HTML 태그에 스타일을 적용하는 선택지이다.
2. **class 선택자**: 여러 요소에 공통 스타일을 적용할 때 사용하는 선택자이다.

3. **id 선택자**: 특정 요소 하나에 스타일을 적용할 때 사용하는 선택자이다.

4. **전체 선택자**: 모든 HTML 요소를 선택하는 선택자이다.

5. **자손 선택자**: 특정 요소 내부에 있는 다른 요소를 선택하는 선택자이다.

## 글꼴, 색상, 배경 등 기초 스타일링

1. 글꼴

- **font-size**: 글자 크기 설정
- **font-family**: 글꼴 종류 설정
- **font-weight**: 글자 굵기 설정

2. 색상

- **color**: 글자 색상 설정

3. 배경

- **backround-color**: 배경 색상 설정
- **backround-image**: 배경 이미지 설정

## Box Model

HTML 요소는 **박스 형태 구조**로 이루어져 있다.

<구성 요소>
- **content**: 실제 내용 영역
- **border**: 요소의 **테두리**
- **padding**: 요소 **안쪽 여백**
- **margin**: 요소 **바깥쪽 여백**
