# GitHub

- 프로그램 코드와 프로젝트 파일을 **인터넷에 저장하고 관리하는 플랫폼**이다.
- **Git** 기반으로 만들어져 **코드 변경 기록을 저장하고 이전 버전으로 되돌릴 수 있다.**
- 여러 개발자가 **하나의 프로젝트를 함께 협업**할 수 있도록 도와준다.

## GitHub의 주요 기능

- **코드 저장**: 프로그램 코드와 프로젝트 파일을 인터넷에 저장할 수 있다.
- **버전 관리**: 코드가 수정될 때마다 변경 기록이 남아 이전 버전으로 되돌릴 수 있다.
- **협업 가능**: 여러 개발자가 하나의 프로젝트를 함께 개발할 수 있다.
- **포트폴리오 가능**: 자신이 만든 프로젝트를 공개하여 다른 사람들에게 보여줄 수 있다.

## GitHub의 구성 요소

- **Repository (레포지토리)**
프로젝트 파일과 코드가 저장되는 공간
- **Commit**
코드의 변경 사항을 기록하는 것
- **Push**
내 컴퓨터의 코드를 GitHub에 업로드하는 것
- **Pull**
GitHub에 있는 코드를 내 컴퓨터로 가져오는 것
- **Branch**
새로운 기능을 개발하거나 수정할 때 사용하는 작업 공간

- 보통 VSCode 같은 코드 편집기와 함께 사용한다.
