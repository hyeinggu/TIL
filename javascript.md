# Javascript

**HTML과 CSS로 구성된 웹 페이지를 동적으로 만들어주는 언어**이다.

### 변수

변수(var)는 우리말로 **변수**라고 하며 **변하는 수**라고 생각하면 이해가 쉽다. 여기서 변하는 것은 **데이터**이다.

- let : **값을 변경할 수 있는** 변수를 선언할 때 사용힌다.
- const : **바뀌지 않는 상수값**을 선언할 때 사용한다.

### Hoisting

**호이스팅(Hoisting)**은 변수 선언과 함수 선언을 **코드의 맨 위로 끌어올려지는** 현상을 일컫는다. 그래서 개발자가 어느 라인 위치에 코드를 선언해도, 실행 되기전 코드가 최상단으로 끌어올려지고 실행되게 된다.

### 조건문

- if : 조건식이 true값을 가질 때 중괄호 안의 실행문이 실행됨, 조건식이 false 값을 가지면 실행되지 않는다.

```
if(조건식) {
    실행문;
    실행문;
    ...
}
```

- else : 조건식이 false면 else 절의 실행문이 실행된다.

```
if(조건식){
    실행문;
} else {
    실행문;
}
```

### 반복문

- for : 조건식이 참인 동안 **주어진 횟수만큼** 실행문을 반복한다.

```
for(초기값; 조건식; 증감식) {
    반복 내용;
}
```

- while : **조건식이 true**일 경우 계속 반복한다.

```
while (조건식) {
    반복 내용;
}
```

### 삼항 연산자

```
(조건) ? value1 : value2
```

조건이 참인 경우 물음표 바로 다음에 있는 value1이 실행되며, 거짓이면 value2가 실행된다.

### 함수 선언

어떤 **기능을 하는 코드를 이름 붙여서 정의**하는 것이다.

### 화살표 함수

`function` 키워드 대신 **화살표(=>)를 사용하여 보다 간략한 방법으로 함수를 선언**할 수 있다.

### 자료구조

- **배열 []**: **여러 개의 값을 저장하고 관리하는 데 사용**되는 자료구조이다.

- **객체 {}** : **여러 데이터를 하나로 묶은** 자료구조이다.
