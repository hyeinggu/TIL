# Javascript

**HTML과 CSS로 구성된 웹 페이지를 동적으로 만들어주는 언어**이다.

### 변수

- let : **값을 변경할 수 있는** 변수를 선언할 때 사용힌다.
- const : **바뀌지 않는 상수값**을 선언할 때 사용한다.

### 조건문

- if : 조건식이 true값을 가질 때 중괄호 안의 실행문이 실행됨, 조건식이 false 값을 가지면 실행되지 않는다.

```
if(조건식) {
    실행문;
    실행문;
    ...
}
```

- else : 조건식이 false면 else 절의 실행문이 실행된다.

```
if(조건식){
    실행문;
} else {
    실행문;
}
```

### 반복문

- for : 조건식이 참인 동안 **주어진 횟수만큼** 실행문을 반복한다.

```
for(초기값; 조건식; 증감식) {
    반복 내용;
}
```

- while : **조건식이 true**일 경우 계속 반복한다.

```
while (조건식) {
    반복 내용;
}
```

### 함수 선언

어떤 **기능을 하는 코드를 이름 붙여서 정의**하는 것이다.

### 화살표 함수

`function` 키워드 대신 **화살표(=>)를 사용하여 보다 간략한 방법으로 함수를 선언**할 수 있다.

### 자료구조

- **배열 []**: **동일한 데이터 타입의 요소들을 연속된 메모리 공간에 저장**하는 방법이다.

- **객체 {}** : **여러 데이터를 하나로 묶은** 자료구조이다.
