# 반응형 & 레이아웃

## 반응형 (Media Query)

- **반응형 웹 디자인** <br>
  화면 요소들을 화면 크기에 맞게 재배치하고 각 요소의 표시 방법만 바꾸어 사이트를 구현해준다.
- **반응형 프로그래밍** <br>
  화면 크기나 환경에 따라 자동으로 모습이 바뀌는 프로그램

반응형의 **장점**

1. 사이트 하나만 만들면 데스크톱 PC, 모바일 모두 사용할 수 있다.
2. 모든 스마트 기기에서 접속 가능하다.
3. 가로 모드에 맞춰 레이아웃 변경 가능하다.
4. 사이트 유지, 관리가 용이하다.

### 조건 종류

- `@media` : 화면 크기나 조건에 따라 CSS를 다르게 적용하는 규칙
- `max-width` : 화면 너비가 이 값보다 작을 때
- `min-width` : 화면 너비가 이 값보다 클 때

### 반응형 단위

- **rem** : r은 root를 의미한다. **항상 루트(html)에 지정된 font-size에 따라서 상대적으로 크기가 결정**된다.
- **%** : **부모 요소의 크기를 기준으로 값이 계산**된다. 정의된 부모 요소가 없다면 기본적으로 루트가 기준이 된다.
- **vw,vh** : 각각 **뷰포트(viewport) 너비와 높이의 1/100**이다. 이때, 뷰포트는 웹 페이지에서 사용자가 볼 수 있는 화면의 영역이며, 뷰포트의 값은 스크롤바를 포함한 값이다.
- **em** : **부모 요소의 font-size를 기준으로 상대적으로 값이 계산**된다. 부모의 font-size에 곱하는 값이라고 생각할 수 있다.

## Flex / Grid

### Flex

FLEXBOX는 **플렉스 컨테이너(Flex Container)와 플렉스 아이템(Flex Item)으로 구성**된다. Flex Container는 Flex Item을 감싸는 **부모 요소**이고, Flex Item은 Flex Container 내에 위치한 **자식 요소**들이다.

- `display: flex` : **Flex Container를 정의하는 기본 속성**이다. 이 속성을 부모 요소에 적용하면 해당 요소의 자식 요소들이 Flex Item으로 변환되어, Flexbox 레이아웃 시스템의 규칙에 따라 배치된다.
- `flex-direction` : Flex Item의 **주 축 방향을 설정**한다. **row가 기본값**이고, Flex Item을 행 방향(왼쪽에서 오른쪽)으로 배치한다.
- `justify-content` : **주 축을 따라 Flex Item을 정렬**한다. **flex-start가 기본값**이고, Flex Item을 주 축의 시작 지점에 정렬한다.
- `align-items` : **교차 축을 따라 Flex Item을 정렬**한다. **stretch가 기본값**이고, Flex Item을 Flex Container의 높이에 맞게 늘린다.

### Grid

`display: grid` : 요소들을 격자 형태로 배치할 수 있게 하는 속성이다. <br>
`grid-template-columns` : 가로(열) 크기와 개수를 설정한다. <br>
`grid-template-rows` : 세로(행) 크기와 개수를 설정한다. <br>

### Gap

gap은 간격이다. 간격은 요소와 요소 사이의 여백을 일정하게 유지해주는 기능으로 반복적인 UI 구조에서 특히 유용하다. 한 번 간격을 정해두면 이후에 요소가 늘어나거나 줄어들어도 전체 균형이 자연스럽게 유지되기 때문에 디자인 관리가 훨씬 쉽다.

- 사이즈 기본: **Hug, Fill, Fixed**
  - **hug**: 내부 콘텐츠 크기에 맞춰 요소가 감싸듯 늘어나는 방식으로 텍스트 길이에 따라 크기가 자연스럽게 바뀐다.
  - **Fill** : 부모 영역의 남은 공간을 채우는 방식으로, 화면 크기에 따라 유연하게 늘어나고 줄어든다.
  - **Fixed** : 너비나 높이를 수치로 지정해 크기를 고정하는 방식으로 어떤 화면에서도 동일한 크기를 유지해야 할 때 사용한다.

### Flex vs Grid 차이<br>

**Flex**는 1차원 레이아웃이며, 가로나 세로 중 하나의 방향으로만 정렬할 수 있다. <br>
**Grid**는 2차원 레이아웃이며, 가로 세로 둘 다 정렬할 수 있고 표처럼 배치가 가능하다.
